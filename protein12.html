PART 01

膜蛋白全景

膜蛋白佔人類所有蛋白的約 30%，是最重要的藥物靶點（~60% 藥物作用於膜蛋白）。 依照嵌入方式分為整合膜蛋白（TM 跨膜螺旋）、外周膜蛋白、脂錨定蛋白三大類。

城堡城牆比喻

細胞膜是城牆，只擋水不擋脂肪。膜蛋白是城牆上的各種設施： GPCR 是訊號旗（有敵情就揮旗傳訊）；Na⁺/K⁺ 幫浦是水車（主動搬水）； 離子通道是吊閘（電壓一到就開閘）；Aquaporin 是管線（讓水安全快速通過）。 四者合作，讓細胞對外有感知、對內有秩序。

PART 02

整合膜蛋白與 GPCR

整合膜蛋白以疏水性跨膜螺旋（TM helix, ~20 個疏水胺基酸）錨定在脂雙層。 GPCR 是其中最大的家族（~800 種），七個 TM 螺旋排列成「桶」， 是人類最重要的藥物靶點（約 34% 的核准藥物作用於 GPCR）。

1

跨膜螺旋的疏水規則｜Hydrophobic TM Helix

約 20 個疏水胺基酸（Leu, Ile, Val, Phe）組成 α-螺旋，螺旋外側為疏水面，完美嵌入脂質雙層的疏水核心。親水的胺基酸側鏈朝向通道內或蛋白質-蛋白質界面。

2

配體結合啟動 G 蛋白｜Ligand-Receptor Activation

配體（荷爾蒙、神經傳遞物、光子）結合 GPCR 胞外端，引發構象變化，傳遞至胞內端，活化與之結合的異源三聚體 G 蛋白（α、β、γ 三個次單元）。

3

第二信使放大訊號｜Second Messenger Amplification

Gα-GTP 擴散，活化下游酵素：Gs 激活 adenylyl cyclase → cAMP↑ → PKA；Gq 激活 PLC → IP₃ + DAG → Ca²⁺釋放 + PKC。一個受體可激活數百個下游分子，訊號放大千倍。

4

β-arrestin 脫敏與回收｜Desensitization

持續刺激後，GRK（G蛋白偶聯受體激酶）磷酸化 GPCR，β-arrestin 結合、阻斷 G 蛋白訊號，並將受體送往核內體回收或降解——這是「受體脫敏」的分子機制。

PART 03

Na⁺/K⁺-ATPase 幫浦與膜電位

Na⁺/K⁺-ATPase 是「主動運輸」的代表。每水解一個 ATP，把 3 個 Na⁺ 打出、2 個 K⁺ 打入， 維持細胞內低 Na⁺、高 K⁺ 的環境，建立膜電位（約 -70 mV）——這是神經和肌肉興奮的能量基礎。

⚡

充電電池比喻

Na⁺/K⁺ 幫浦就像在為電池充電。它持續把 Na⁺（正電）打出、K⁺（正電）打入， 製造一個跨膜電位差（約 -70 mV，內負外正）。這個「電量」讓神經可以發送電訊號—— 神經動作電位就是短暫「放電」再「重新充電」的過程，Na⁺/K⁺ 幫浦是那個充電器。

E1

E1 構象：高親和 Na⁺｜Na⁺ Binding Inside

幫浦蛋白（α 次單元）在細胞內側以高親和力結合 3 個 Na⁺；ATP 結合於磷酸化結構域，準備水解。

E2

ATP 水解與磷酸化：Na⁺ 釋放細胞外｜Phosphorylation

ATP 水解，磷酸基轉移到 Asp369（E1P → E2P），構象變化，Na⁺ 親和力下降，3 個 Na⁺ 釋放到細胞外。

E3

E2 構象：高親和 K⁺｜K⁺ Binding Outside

E2P 在細胞外側以高親和力結合 2 個 K⁺，觸發去磷酸化（Pi 釋放），構象轉回 E1，K⁺ 釋放到細胞內。

PART 04

離子通道與電壓門控

離子通道是「被動運輸」：順著電化學梯度讓離子通過，不需要 ATP。 電壓門控通道（Voltage-Gated Channel）依膜電位開關，是神經動作電位的核心分子機器。 每秒可通過 10⁷ 個離子，速度比載體蛋白快 1,000 倍以上。

①

S4 電壓感測器｜Voltage Sensor

每個 TM domain 的第四段（S4）富含帶正電的 Arg/Lys，膜電位去極化時，靜電力驅動 S4 向胞外移動，引發構象變化，pore gate 開啟。去極化速度決定 Nav（快）vs Kv（慢）的時間差異。

②

選擇性過濾器｜Selectivity Filter

通道最窄處（selectivity filter）以精確尺寸和靜電環境篩選離子。K⁺ 通道的 GYG 序列（鉀通道標誌）只讓 K⁺ 通過（完全去水化後直徑 2.66 Å），而 Na⁺（2.32 Å）因去水化能量太高而被排除。

③

水通道 AQP 的嚴格水選擇性｜Aquaporin Selectivity

AQP 的 NPA 基序（Asn-Pro-Ala，對稱出現兩次）形成「砂漏」結構，中央正電荷阻擋 H⁺（防止酸鹼干擾），ar/R 過濾器排除所有非水分子，每秒可通過 3×10⁹ 個水分子，卻完全不透其他離子。

PART 05

膜蛋白失調與疾病

膜蛋白功能異常影響範圍極廣：從心律不整、高血壓到神經疾病和癌症， 幾乎所有重大疾病都有膜蛋白參與。

治療策略

通道阻斷劑：局部麻醉藥（利多卡因）阻斷 Nav；鈣離子阻斷劑（維拉帕米）阻斷 Cav，用於心律不整。
GPCR 靶向藥物：β 受體阻斷劑（心臟病）、抗組胺藥（H1-GPCR）、奧氮平（D2-GPCR 抗精神病）——34% 的上市藥物都靶向 GPCR。
基因治療：KATP 通道突變（先天性高胰島素血症）可用磺醯脲類藥物（SUR1 結合）部分補救。

🧱 膜結構 Membrane Structure
PL 磷脂雙層Phospholipid Bilayer 膜結構▶

細胞膜的基本骨架。磷脂分子有「頭」（磷酸基，親水）和「尾」（脂肪酸鏈，疏水）。兩層磷脂尾對尾排列，形成中間疏水、兩側親水的薄膜（~7nm），只允許小的非極性分子自由穿越。

膜組成：磷脂（50%）+ 蛋白質（40%）+ 膽固醇（10%）

磷脂雙層像三明治：外面兩片麵包（親水頭）夾住中間的奶油（疏水尾），油水分離。

TM 跨膜螺旋Transmembrane Helix 膜結構▶

~20 個連續疏水胺基酸（Leu, Ile, Val, Ala, Phe）組成 α-螺旋，外側面皆為疏水胺基酸，正好匹配脂雙層 3 nm 的疏水厚度，將蛋白質錨定在膜上。單次跨膜（Type I/II）或多次跨膜（GPCR 7次，AQP 6次）。

疏水長度判斷：連續 17–25 個疏水胺基酸 → 預測跨膜螺旋（Kyte-Doolittle 疏水性分析）

跨膜螺旋就像插入黃油的牙籤——只有完全疏水的牙籤才能在黃油（脂雙層）裡穩定存在。

MΨ 膜電位Membrane Potential 電生理▶

膜兩側的電位差，靜息時約 -70 mV（內負外正），由 Na⁺/K⁺ 幫浦維持的離子不均衡所建立。膜電位是神經訊號（動作電位）、肌肉收縮、胰島素分泌等生理功能的能量儲存形式。

Nernst 方程式：E = (RT/zF) × ln([離子]外/[離子]內)

膜電位就像繃緊的彈弓，平時蓄力（-70 mV），一旦觸發就彈出訊號（動作電位），再慢慢重新拉緊（復極化）。

📡 信號受體 Signal Receptors
GPCR G 蛋白偶聯受體G Protein-Coupled Receptor 受體▶

七次跨膜 α-螺旋受體家族，人類約 800 種。配體結合後活化胞內 G 蛋白（Gs/Gi/Gq），啟動 cAMP / IP₃ / DAG 等第二信使。調控視覺、嗅覺、心率、免疫、代謝等幾乎所有生理功能。

配體 → GPCR → Gα-GTP → Adenylyl cyclase → cAMP → PKA → 磷酸化目標

GPCR 是細胞的「智慧門鈴」——鈴聲（配體）響了，裡面的機器（G蛋白/cAMP）就啟動，不需要直接進門。

β-ar β-arrestin 脫敏機制Beta-Arrestin Desensitization 調控▶

持續刺激 GPCR 後，GRK（G蛋白偶聯受體激酶）磷酸化 GPCR 胞內端，β-arrestin 結合、遮蔽 G 蛋白結合位、終止訊號，並引導受體進入內吞囊泡。β-arrestin 也能獨立傳遞「偏向信號」（biased agonism），是新藥設計方向。

GPCR 刺激 → GRK 磷酸化 → β-arrestin 結合 → G蛋白訊號終止 → 受體內吞
⚙️ 主動運輸 Active Transport
NKA Na⁺/K⁺-ATPaseSodium-Potassium Pump 幫浦▶

P 型 ATPase，每水解一個 ATP，逆濃度梯度將 3 個 Na⁺ 泵出細胞、2 個 K⁺ 泵入細胞，建立並維持靜息膜電位和細胞體積。消耗細胞約 20–30% 的 ATP（神經細胞更高）。α 次單元是強心苷（毛地黃）的作用靶點。

3 Na⁺(細胞內) + 2 K⁺(細胞外) + ATP → ADP + Pi + 3 Na⁺(外) + 2 K⁺(內)

Na⁺/K⁺ 幫浦是細胞的「背景音樂」——你感覺不到它，但它一刻不停，讓每個細胞保持準備好發訊號的狀態。一旦停止（缺氧、地高辛中毒），細胞會腫脹、神經失常。

SERCA 肌質網 Ca²⁺-ATPaseSarco/Endoplasmic Reticulum Ca²⁺-ATPase 幫浦▶

P 型 ATPase，每水解一個 ATP 將 2 個 Ca²⁺ 泵回肌質網（SR）或內質網（ER），使細胞質 Ca²⁺ 回到靜息水平（~100 nM），終止肌肉收縮或信號。SERCA 抑制劑 Thapsigargin 是研究 Ca²⁺ 信號的常用工具。

2 Ca²⁺(細胞質) + ATP → 2 Ca²⁺(SR/ER) + ADP + Pi
💧 被動通道 Passive Channels
VGC 電壓門控通道Voltage-Gated Ion Channel 通道▶

由膜電位變化控制開關的離子通道。Nav（鈉通道）負責動作電位上升相（去極化），Kv（鉀通道）負責下降相（復極化），Cav（鈣通道）負責肌肉收縮信號和突觸囊泡釋放。

動作電位：靜息(−70mV) → 去極化(Nav開，+40mV) → 復極化(Kv開) → 過極化(−90mV) → 靜息

Nav 和 Kv 就像接力賽的兩個選手——Nav 先衝（Na⁺ 衝入），Kv 接棒（K⁺ 衝出），共同完成一次動作電位，然後幫浦重設場地。

LGC 配體門控通道Ligand-Gated Ion Channel 通道▶

神經傳遞物結合胞外端後直接開啟離子通道（「直接偶聯」，比 GPCR 快 1000 倍）。nAChR（菸鹼型乙醯膽鹼受體）是代表：5 個次單元圍成通道，乙醯膽鹼結合後 Na⁺ 快速內流，突觸後膜去極化。

nAChR: (α₂βγδ) 五聚體 → ACh 結合 2α → 開孔 → Na⁺/K⁺ 流動 → 去極化

LGC 是「鑰匙插入立即開門」（配體 = 鑰匙，通道 = 門），GPCR 是「按門鈴後門裡的人去開門」——前者快，後者可放大。

AQP 水通道蛋白Aquaporin 通道▶

高度選擇性的水通道，每秒允許 3×10⁹ 個水分子通過，而完全阻擋離子（包括 H⁺）。砂漏結構中央的兩個 NPA 基序形成正電位，阻止帶正電的質子（H⁺）通過——這保護細胞的酸鹼平衡。人類有 13 種（AQP0–AQP12）。

AQP 核心：NPA-box × 2（正電排斥 H⁺）+ ar/R 過濾器（尺寸篩選）

AQP 是細胞膜上的「純水管道」——只讓水（不帶電、極小）通過，就連電荷最小的離子（H⁺）也被靜電場擋在外面。2003 年諾貝爾化學獎（Peter Agre）。

Kir 內向整流鉀通道Inwardly Rectifying K⁺ Channel 通道▶

在靜息電位時允許 K⁺ 流入（內向整流），維持靜息膜電位接近 K⁺ 平衡電位。KATP（ATP 感測型 Kir）連接細胞代謝與膜電位：ATP 高（飽食）→ 通道關閉 → 去極化 → 胰島素分泌。磺醯脲類藥物（第一代糖尿病藥）阻斷 KATP 促進胰島素分泌。

KATP 型：[ATP]內高 → 關閉通道 → 膜去極化 → Cav 開啟 → Ca²⁺ 流入 → 胰島素囊泡釋放
🔬 功能補充 Key Concepts
AP 動作電位Action Potential 電生理▶

神經元或肌肉細胞的「電訊號」。去極化達閾值（約 -55 mV）時，Nav 快速開啟，Na⁺ 大量內流使膜電位升至 +40 mV；Kv 延遲開啟，K⁺ 外流使膜電位復極化回到 -70 mV（甚至短暫過極化 -90 mV）。動作電位「全有全無」。

閾值(−55mV) → Nav 開 → Na⁺ 內流(+40mV) → Kv 開 → K⁺ 外流 → 復極化(−90mV) → 靜息(−70mV)

動作電位就像倒塌的骨牌——一旦開始（超過閾值），後面的骨牌（Nav → Kv）必然依序倒下，無法「半途而廢」。

2MS 第二信使系統Second Messenger System 信號▶

GPCR 活化後在細胞內產生的小分子信號物質，負責放大和傳遞第一信使（配體）的訊號。主要有三類：cAMP（由 adenylyl cyclase 合成，激活 PKA）、IP₃/DAG（由 PLC 分解 PIP₂，IP₃ 釋放 ER Ca²⁺，DAG 激活 PKC）、Ca²⁺（調控多種下游效應器）。

Gs → AC → cAMP → PKA | Gq → PLC → IP₃ + DAG → Ca²⁺ + PKC

第二信使就像廣播中繼站——配體（一個分子）活化一個 GPCR，GPCR 活化 10 個 G 蛋白，每個 G 蛋白產生 100 個 cAMP，每個 cAMP 激活 PKA...訊號放大千萬倍。

Nst Nernst 方程式Nernst Equation 電化學▶

計算單一離子「平衡電位」（equilibrium potential）的方程式——當電化學梯度使離子恰好停止淨流動時的膜電位。K⁺ 平衡電位約 -90 mV（靜息電位接近此值）；Na⁺ 平衡電位約 +60 mV（動作電位峰值接近此值）。

E = (RT/zF) × ln([離子]外/[離子]內)
室溫：E(K⁺) ≈ -90 mV | E(Na⁺) ≈ +60 mV | E(Ca²⁺) ≈ +135 mV
